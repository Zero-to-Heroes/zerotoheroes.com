You can use Markdow syntax to format your text (bold, italics, paragraphs, etc)

Additionally, you can:

  • Add a link to a card name using [[The Card Name]]. This will create a hyperlink that will display the card on hover
  • Add a link to a deck hosted on HearthPwn.com using [http://www.hearthpwn.com/decks/your_deck_id]. This will create a hyperlink that will display the cards list on hover
  • Use the timestamp icon to create a link to a specific moment in the video. You can freely adjust the timestamp figures to match exactly what you need
  • Use the slow motion icon to create a timestamp and make the video play in slow motion once the user clicks on it. You can change how much the video is slowed by changing the 0.5 figure after the s (the lower, the slower the video)
  • Use the loop icon to create a slow motion link, and make a one-second-long sequence that repeats endlessly. You can change the duration of the loop by changing the value after the L (L1.5 would mean a 1.5s loop)
  • Use the drawing icon to create a timestamp with a drawing. Once you click on the icon, you will enter drawing mode. Simply click and hold on the white canvas on the video to draw anything you like, and click again on the icon to save. If you don't like what you just drew, simply erase the [tmp0] part in the comment, and start again!
  • Use the side-by-side video icon to create a sequence with side-by-side videos, to compare two clips to show a difference. You can of course combine it with the other tools provided above (by default a sequence is created with loop and slow motion)
